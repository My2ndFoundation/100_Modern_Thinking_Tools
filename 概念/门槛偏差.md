## 一句话定义
你看到的都是跨过某个门槛之后的样本——两个原本不相关的变量加和后设门槛，门槛内样本就显得负相关；又名 [[伯克森悖论]]。

## 简介
门槛偏差的学术正名是伯克森悖论（Berkson's paradox，亦称 Berkson's bias 或 collider bias），由生物统计学家 Joseph Berkson 于1946年在《Biometrics Bulletin》论文"Limitations of the Application of Fourfold Table Analysis to Hospital Data"中首次系统描述。（[Berkson's paradox - Wikipedia](https://en.wikipedia.org/wiki/Berkson%27s_paradox)）该悖论的核心机制是：当研究样本通过一个共同的门槛条件被选出时，原本独立的两个变量在该子样本中会呈现虚假的负相关，纯粹是选择偏差造成的统计幻觉，并非真实世界中的规律。（[Berkson's bias, selection bias, and missing data - PMC](https://pmc.ncbi.nlm.nih.gov/articles/PMC3237868/)）

## 详细解释
[[选择偏差]] 第四类，正式名称是 [[伯克森悖论]]（Berkson's Paradox）。Joseph Berkson 于1946年在《Biometrics Bulletin》首次描述；Westreich 2012 是后续重要综述。<!-- 溯源订正 2026-05-15: 原作「Westreich 2012 是经典综述」未提及 Berkson 1946 原始文献，据 https://en.wikipedia.org/wiki/Berkson%27s_paradox 补充原始出处 -->

机制：两个原本可能完全不相关的变量 X 和 Y，如果你把它们加在一起设定一个总分门槛 X+Y ≥ T，然后只看跨过门槛的样本，就会发现 X 和 Y 在样本里显得负相关——纯粹是门槛伪造出的规律。

万维钢的婚恋市场示例：
- 假设女生看重的男生价值 = 颜值 + 人品。
- "备胎门槛"卡掉颜值 + 人品过低的男生。
- 门槛内：两项都高的极少（被早早抢走），两项都低的没进来——剩下大多是一项高一项低。
- 结论错觉：颜值与人品互斥（"长得帅的男人都是渣男"）。

同理可破：
- 业务能力强的人情商低 → 公司招聘门槛是"业务+情商"达标。
- 美女通常脾气差。
- 体育好的学生文化课差。

万维钢金句：「相亲不是人口普查，招聘不是社会抽样，门槛会制造对立，筛子会伪造规律。」
