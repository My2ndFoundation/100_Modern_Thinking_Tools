## 一句话定义
[[约翰·凯利]] 1956 公式：在乘法、可重复的世界里，最佳下注比例 f* = (bp − q)/b = [[Edge]] / odds——把认知优势翻译成下注规模。

## 来源
- [[凯利公式：乘法世界里的认知变现]]
- [[非遍历性：玩家怕方差，庄家爱方差]]

## 何时使用
- 任何"乘法资产 + 可重复 + 不确定"的场景：投资、职业、健康、信任。
- 当你需要决定"出手力度"而不是"出不出手"时。

## 操作步骤
1. 估 p（你认为的胜率）和 b（赔率）。
2. 算 [[Edge]] = bp − q（其中 q = 1 − p）。Edge ≤ 0 → 不下注。
3. 算 f* = edge / b。
4. 估计有误差时用半凯利或分数凯利。
5. 永远不要 all-in（p < 1）；保留止损机制。
6. 单次成败不重要——追求长期复利。

## 例子
- 胜率 0.6、赔率 1，凯利说押 20%。
- 袁绍/曹操官渡战例：edge 大却不出手 = "干大事而惜身"。
- 巴菲特"不买看不懂的股票" = edge 不明就不下注。
- 单细胞生物的 [[表型赌注对冲]]：休眠比例 = 凯利未下注比例。

## 相关工具
- [[能耐寻求]]——保留下次下注能力 = 保留选项。
- [[OODA 环]]——每轮 OODA 是一次小凯利下注。
- [[杠铃策略]]——凯利在 [[非遍历性]] 风险下的简化变体（f* ≈ 10%）。
- [[风险共担]]——凯利的集体版：把多个个体的下注合并为庄家。
